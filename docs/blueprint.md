# **App Name**: Rápido y Rico

## Core Features:

- Visualización del Menú (Cliente): Muestra el menú del food truck con nombre, precio, foto, descripción y disponibilidad.
- Registro de Pedido (Cliente): Permite al cliente agregar productos al pedido, ingresar un apodo y confirmar el pedido, recibiendo un número de pedido único.
- Visualización de Pedidos en Tiempo Real (Vendedor): Presenta los pedidos realizados por los clientes en tiempo real, mostrando número de pedido, productos, hora y estado (Pendiente, En preparación, Listo).
- Actualización del Estado del Pedido (Vendedor): Permite al vendedor cambiar el estado de cada pedido (Pendiente, En preparación, Listo).
- Panel de Control de Menu: Admin view for control and edition of food items including name, description, image url, price, and availability.
- Generación de Apodos Sugeridos: Tool to suggest nicknames based on user food preferences. Using user input to generate creative nicknames. Tool should ensure to follow a validation step.
- Notificaciones: Notificaciones visuales y sonoras para alertas al vendedor cuando entra un pedido.

## Style Guidelines:

- Color primario: Naranja brillante (#FF9800) para reflejar la energía y el apetito.
- Color de fondo: Gris claro (#F5F5F5) para un aspecto limpio y moderno.
- Color de acento: Amarillo (#FFEB3B) para destacar elementos importantes y acciones.
- Fuente para títulos y cuerpo: 'Poppins' (sans-serif) para una apariencia geométrica y contemporánea.
- Iconos: Utilizar iconos claros y modernos que representen las diferentes categorías de alimentos.
- Diseño: Priorizar un diseño responsive y fácil de usar en dispositivos móviles y tablets.
- Animaciones: Implementar animaciones sutiles al agregar productos al carrito o actualizar el estado del pedido.
